Gadget Marketplace - Milestone 1
--------------------------------
Project Overview: Static SPA frontend + backend folder structure (DAO + config) + SQL schema placeholder.
How to run locally:
1) Place the folder in C:\xampp\htdocs\gadget-marketplace
2) Start Apache & MySQL in XAMPP, create DB 'gadget_marketplace'
3) Import database/database.sql in phpMyAdmin
4) Visit /frontend/index.html for UI, /backend/test.php to test DB connection
Folder Structure:
  - frontend/ (index.html, style.css, script.js)
  - backend/ (config/, dao/, test.php)
  - database/ (database.sql)
